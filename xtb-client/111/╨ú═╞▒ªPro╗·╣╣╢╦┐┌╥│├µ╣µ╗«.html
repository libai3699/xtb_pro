页面规划说明
底部导航栏设计
根据微信小程序底部导航栏tabbar规范，校推宝Pro机构端口设计了5个核心功能页面，完全符合微信小程序的设计规范。每个页面都对应机构端的核心业务流程，确保用户可以快速访问所需功能。
智能看板（首页）
智能看板作为机构端首页，提供核心业务指标概览和实时动态提醒，帮助机构管理者快速了解当前业务状况。
✓
推广中的活动数量、参加推广的校园代理人数、活动报名人数、当日活动签到人数等核心指标展示
✓
新增报名学生提醒、新增代理提醒、活动节点提醒等实时动态流
✓
数据实时更新，确保管理者获取最新业务信息
代理管理
代理管理模块提供校园代理的全生命周期管理，包括代理审核、状态管理和数据统计。
✓
按学校/活跃度筛选代理列表，显示基础信息（姓名、学校、状态、学院、专业、年级）
✓
代理审核流程，支持申请审核和状态管理
✓
代理数据统计，包括代理总数、活跃代理数、推广活动数量和带来参加活动的总人数
任务中心
任务中心是机构创建和管理活动的核心模块，提供活动创建、管理和基础监控功能。
✓
基础活动创建，包括活动基本信息、佣金规则、目标人群、时间设置和参与信息收集
✓
活动管理，支持活动列表展示和状态管理
✓
基础监控，包括参与人数统计、佣金支出统计和参加活动的学生清单
线索管理
线索管理模块帮助机构高效管理潜在客户信息，提高转化效率。
✓
线索池管理，展示所有线索列表并提供基础筛选功能
✓
防重复检测，通过手机号去重和基础清洗机制确保数据质量
✓
线索来源追踪，帮助机构分析不同渠道的获客效果
财务管理
财务管理模块提供财务数据统计和核对功能，帮助机构掌握营销投入产出情况。
✓
财务统计数据，包括总预算、总佣金、已核对金额和待核对金额
✓
代理财务统计，展示代理佣金总额、已核对金额和待核对金额
✓
用户分享奖励统计，包括奖励总额统计和奖励明细查询
✓
注：P0版本财务数据仅用于各端口进行核对统计，不包含打款及提现相关功能
